# 报告详细模板

本文件为 community-survey-analysis 技能的参考资料，包含报告各章节的具体写法与示例。

## 核心摘要模板

> 通过对[社区名]居民（[身份描述]）的深度访谈发现：社区呈现出"[核心特征1] + [核心特征2] + [核心特征3]"的[模式名称]特征。[制度/筛选机制]使居民构成[特征]，[关键发现概述]。[与已有调研的对比结论]。[主要改进诉求]。

**要素清单：**
- 社区名 + 受访者身份
- 3-5个核心特征关键词（用"+"连接）
- 制度筛选效果
- 最重要的发现（1-2句）
- 与其他社区的对比要点（如有）
- 主要痛点与诉求

## 受访者画像表模板

| 社区 | 编号 | 身份 | 居住时长 | 家庭特征 |
|------|------|------|----------|----------|
| XX社区 | 受访者1 | [身份]（[性别]，[特征]） | [时长] | [家庭描述] |

**填写规范：**
- 身份：区分"员工/家属/非员工"等
- 居住时长：精确到年或月，如"近3年""2年""较短（未明确）"
- 家庭特征：有无子女、子女年龄段、是否常住

## 核心发现各维度写法

### 维度1: 制度筛选与进入方式

**必须覆盖：**
- 受访者对入住规则的认知程度（清晰/模糊/不知）
- 规则感知是否存在信息不对称
- 受访者对筛选机制的态度（积极/中性/消极）

**分析角度：**
- 与其他社区/受访者对比规则感知差异
- 制度筛选与居民满意度的关系

### 维度2: 空间边界与日常行为

**必须覆盖：**
- 通勤模式（班车/自驾/公交/地铁）
- 出行范围（社区内/园区/市区）
- 时间分配（工作/休息/家庭/社交）
- 与城市公共交通的连接程度

**分析角度：**
- "内缩式"生活模式 vs 城市融合
- 通勤时间节省带来的生活质量效应

### 维度3: 资源依赖与生活配套

**必须覆盖：**
- 购物渠道（线上/线下/社区内/外部）
- 餐饮解决方式（食堂/外卖/自煮/外出）
- 就医便利度
- 兴趣/教育配套（如有子女）

**分析角度：**
- "假性自足"程度（表面满足 vs 实际依赖外部）
- 不同家庭结构的配套需求差异

### 维度4: 社交网络与邻里互动

**必须覆盖：**
- 社交圈构成（同事/邻居/外部朋友）
- 邻里互助频率（有/无/极少）
- 替代机制（物业/自己解决/其他）
- 社区活动参与度

**分析角度：**
- "熟人照护网络"是否形成
- 物业对邻里互助功能的替代效应

### 维度5: 子女教育与成长环境

**必须覆盖：**
- 教育是否为选址动因
- 对子女社交环境的评估
- 课余安排方式
- 对成长视野受限的担忧程度

**分析角度：**
- 同龄儿童群体的"自然形成"优势
- 代际锁定的强度

### 维度6: 心理归属与身份认同

**必须覆盖：**
- "家"vs"临时居所"的自我定位
- 身份认同感（企业身份/城市身份）
- 归属感的来源分析
- 入住预期与实际感受的匹配度（如有提及）

**分析角度：**
- 制度筛选对归属感的影响（正面/负面）
- 家庭结构对归属感的调节作用
- 居住时长作为归属感的时间锚点

### 维度7: 搬离成本与锁定机制

**必须覆盖：**
- 搬离时最舍不得的事物
- 是否因居住改变过工作/生活选择
- 锁定因素的具体构成（环境/通勤/子女/社交/其他）
- 对搬离的态度（抗拒/不舍/淡然/期待）

**分析角度：**
- 区分"结构性锁定"（子女教育等不可逆因素）vs "生活惯性"（可替代的偏好）
- 锁定效应的多重因素叠加
- "锁定的隐性成本"——搬离时对子女入学衔接、照护网络连续性的影响

### 维度8: 改进建议

**必须覆盖：**
- 对社区开放度的态度（保持/扩大/收缩）
- 最希望优化的具体方面
- 对现状的总体满意度
- 其他具体建议

## 深度洞察写法

深度洞察应超越单维度描述，提炼跨维度的模式或矛盾。常见模式：

1. **分异效应**：同一物理空间内，不同群体（有子女/无子女）的体验截然不同
2. **双面效应**：同一机制（如制度筛选）同时产生正面和负面效果
3. **共性缺失**：多个社区共同缺乏的要素（如邻里互助、社区活力）
4. **隐性成本**：表面不显著但实际重要的代价（如搬离对子女教育的影响）

## 结论建议写法

建议应区分受众：

```
1. 对[群体A]：[具体建议]
2. 对[群体B]：[具体建议]
3. 社区A专项：[针对性建议]
4. 社区B专项：[针对性建议]
5. 共性需求：[通用建议]
```
